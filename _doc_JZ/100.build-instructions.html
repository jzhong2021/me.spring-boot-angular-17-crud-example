Build and Run Instructions

Spring Boot + Angular 17 CRUD Example

Prerequisites

Before building and running the project, ensure you have the following installed:

  • Java 17 (Spring Boot requires Java 17)
  • Maven (for building Spring Boot)
  • Node.js and npm (for Angular)

Build and Run Steps

1. Build and Run Spring Boot Backend

Open a terminal in the spring-boot-server directory:

cd spring-boot-server
mvn spring-boot:run
.\mvnw.cmd spring-boot:run (JZ: Maven Wrapper)

The Spring Boot server will start on port 8080 (default). The API will be available at http://localhost:8080/api/tutorials.

Note: The README mentions port 8081, but the Angular service is configured to use port 8080, so use 8080.
2. Build and Run Angular Frontend

Open a new terminal in the angular-17-client directory:

cd angular-17-client
npm install
ng serve
npm start (JZ: optional, npm start is the same as ng serve)

The Angular app will start on port 4200 (default) and be available at http://localhost:4200/.

Note: The README suggests ng serve --port 8081, but the Angular service points to http://localhost:8080, so the frontend can run on any port (default 4200 is fine).
3. Access the Application
  • Frontend: http://localhost:4200/
  • Backend API: http://localhost:8080/api/tutorials
  • H2 Database Console: http://localhost:8080/h2-ui (if needed)

Quick Summary

  1. Start Spring Boot: cd spring-boot-server && mvn spring-boot:run
  2. Start Angular: cd angular-17-client && npm install && ng serve
  3. Open browser: http://localhost:4200/

The Angular frontend will communicate with the Spring Boot backend on port 8080.

Project Structure

  • spring-boot-server/ - Spring Boot backend application
  • angular-17-client/ - Angular 17 frontend application

Technologies Used

  • Backend: Spring Boot 3.1.0, Spring Data JPA, H2 Database
  • Frontend: Angular 17, Bootstrap 4.6.2

Angular routerLink & Export Component – Q&A

1. Original Question

User: I have used Angular CLI to create a new Component export, so I need to add something like the following into app.component.html, but I don't understand how it works. Please explain in English and Chinese.

<a routerLink="add" class="nav-link">Add</a>

2. Explanation of routerLink

English

In Angular, this line:

<a routerLink="add" class="nav-link">Add</a>

creates a navigation link that tells the Angular router to go to the route whose path is "add" when the link is clicked.

  • <a> tag: a standard HTML anchor element.
  • routerLink="add":
    • Angular directive (not plain HTML).
    • Navigates to the route with path: 'add' without reloading the whole page.
    • The route is defined in the routing configuration, for example: 
      { path: 'add', component: AddTutorialComponent }
  • class="nav-link": CSS class for styling only; it doesn’t affect routing logic.

The component for a route is rendered where you place:

<router-outlet></router-outlet>
中文

在 Angular 中，这行代码：

<a routerLink="add" class="nav-link">Add</a>

定义了一个导航链接，点击时会让 Angular 的路由器（Router）跳转到路径为 "add" 的路由。

  • <a> 标签：普通的 HTML 超链接元素。
  • routerLink="add"：
    • 这是 Angular 的指令（directive），不是普通 HTML 属性。
    • 点击时不会整页刷新，而是在单页应用内部通过 Router 进行跳转。
    • 会匹配到路由配置中的 path: 'add'，例如： 
      { path: 'add', component: AddTutorialComponent }
  • class="nav-link"：用于样式的类名，与路由行为无关。

路由对应的组件会渲染在：

<router-outlet></router-outlet>

这个“路由出口”里。

3. Using ExportComponent as a routed page (solution #1)

English – Steps
  1. Add an export route in app-routing.module.ts that points to ExportComponent.
  2. Add a navigation link in app.component.html using routerLink="export".
  3. Keep <router-outlet> so routed components are displayed correctly.

3.1 Routing configuration

import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { TutorialsListComponent } from './components/tutorials-list/tutorials-list.component';
import { TutorialDetailsComponent } from './components/tutorial-details/tutorial-details.component';
import { AddTutorialComponent } from './components/add-tutorial/add-tutorial.component';
import { ExportComponent } from './components/export/export.component';

const routes: Routes = [
  { path: '', redirectTo: 'tutorials', pathMatch: 'full' },
  { path: 'tutorials', component: TutorialsListComponent },
  { path: 'tutorials/:id', component: TutorialDetailsComponent },
  { path: 'add', component: AddTutorialComponent },
  { path: 'export', component: ExportComponent }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }
中文说明

在 app-routing.module.ts 中：

  • 引入 ExportComponent。
  • 在 routes 数组中新增：{ path: 'export', component: ExportComponent }。
  • 这样访问 /export 时，就会渲染 ExportComponent。

    <a href="#" class="navbar-brand">bezKoder</a>

        <a routerLink="tutorials" class="nav-link">Tutorials</a>

      <li class="nav-item">
        <a routerLink="add" class="nav-link">Add</a>

      <li class="nav-item">
        <a routerLink="export" class="nav-link">Export</a>
      </li>
    </div>
  </nav>

  <div class="container mt-3">
    <router-outlet></router-outlet>
  </div>
</div>
中文说明

在 app.component.html 中，导航栏增加：

  • <a routerLink="export" class="nav-link">Export</a> 用于跳转到 /export。
  • <router-outlet></router-outlet> 作为路由出口保持不变。

Spring Boot Default Port - Q&A

Question:

So, 8080 is the default port for Spring Boot, right?

Answer:

Yes, that's correct! 8080 is the default port for Spring Boot applications.

Default Port: If you don't specify a port in your configuration, Spring Boot will automatically use port 8080.
Question:

Spring Boot 和 Tomcat 都使用默认端口 8080，有什么理由吗？

(Why do Spring Boot and Tomcat use the same default port 8080?)

Answer:

因为 Spring Boot 内嵌了 Tomcat！

Spring Boot 默认使用 embedded Tomcat 作为其 Servlet 容器。所以它们实际上是同一个服务器：

  • Tomcat - 独立的 web 服务器/servlet 容器
  • Spring Boot - 将 Tomcat 嵌入到应用程序中
  • 当你运行 Spring Boot 时，实际上是在运行内嵌的 Tomcat
为什么是 8080？

8080 已经成为 Java web 应用服务器的事实标准端口。历史上：

  • 端口 80 是 HTTP 的标准端口（需要管理员权限）
  • 8080 是常用的替代端口（不需要特殊权限）
  • 所有主流 Java 服务器（Tomcat、Jetty、Undertow）都采用 8080

其他可选的嵌入式服务器：

  • Jetty - 默认端口也是 8080
  • Undertow - 默认端口也是 8080

How to Change the Port

If you want to change from the default port 8080 to another port (like 8081), you need to modify the following:

1. Spring Boot Configuration

Add the port configuration in application.properties:

server.port=8081

File location: spring-boot-server/src/main/resources/application.properties

2. Angular Service Configuration

Update the API base URL in tutorial.service.ts:

const baseUrl = 'http://localhost:8081/api/tutorials';

File location: angular-17-client/src/app/services/tutorial.service.ts

Change from: http://localhost:8080/api/tutorials

Change to: http://localhost:8081/api/tutorials

3. CORS Configuration (if needed)

Ensure the CORS configuration in TutorialController.java allows your Angular frontend:

@CrossOrigin(origins = "http://localhost:4200")

Or if you want to allow the backend port:

@CrossOrigin(origins = "http://localhost:8081")

Summary

  • Default Port: Spring Boot uses 8080 by default
  • To Change: Add server.port=8081 in application.properties
  • Don't Forget: Update the Angular service URL to match the new port
  • After Changes: Restart both Spring Boot and Angular applications

Default Ports Summary

Application Default Port Configuration File
Spring Boot 8080 application.properties
Angular (ng serve) 4200 package.json or command line
Tomcat (embedded) 8080 application.properties
Note: After making port changes, you'll need to:
  1. Stop the Spring Boot server (Ctrl+C)
  2. Restart it: .\mvnw.cmd spring-boot:run
  3. Restart Angular: npm start
  4. Access the new port: http://localhost:8081/api/tutorials
